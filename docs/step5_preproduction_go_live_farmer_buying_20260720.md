## 1. Objectif de l'étape 5

L'étape 5 transforme la procédure de recette terrain en un plan opérationnel de pré-production.

Le but n'est pas encore de lancer l'application en production réelle, mais de préparer un lancement maîtrisé avec :

- des utilisateurs clairement identifiés ;
- des téléphones de test ;
- des villages / RT / producteurs test ;
- des règles GO / NO-GO ;
- une procédure de retour arrière ;
- un registre d'incidents ;
- une décision finale du Branch Manager avant
